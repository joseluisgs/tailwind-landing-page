Uso

Usar los siguientes comandos.

// Instalar dependencias
npm install

// Comenzar en modo desarrollo
npm run dev

// Generar ficheros de producción
npm run prod

Opciones

Puedes cambiar la configuración de tus ficheros desde el fichero config.js

config: { ... port: 9050 //Puerto de escucha }, paths: { root: "./", src: { base: "./src", css: "./src/css", js: "./src/js", img: "./src/img" }, dist: { base: "./dist", css: "./dist/css", js: "./dist/js", img: "./dist/img" }, build: { base: "./build", css: "./build/css", js: "./build/js", img: "./build/img" } }
